## Overview
Hackathon HUB is a project designed to help hackathon teams manage ideas,
organize tasks, and collaborate efficiently.

## Problem Statement
Hackathon teams often struggle with coordination, task tracking,
and idea management under time pressure.

## Solution
This project provides a simple structure to:
- Organize ideas
- Track progress
- Improve collaboration

## Tech Stack
- Python
- Git & GitHub
